Authentication Authenticate a user

Register User /admin/api/register Private POST

Private: This endpoint is private. You must be authenticated to use it.

Description

Register a user.


Parameters

NameTypeDescription
usernamestringThe username of the user to register.
passwordstringThe password of the user to register.

Responses

CodeTypeDescription
4014xx Client ErrorOnly authenticated users can use this endpoint.

Example

<form action="/admin/api/register" method="POST">
    <input type="text" name="username" />
    <input type="password" name="password" />
</form>

Login User /admin/api/login POST

Note: This endpoint always redirects the user, regardless of input.

Description

Log in a user.


Parameters

NameTypeDescription
usernamestringThe username of the user to log in.
passwordstringThe password of the user to log in.

Responses

CodeTypeDescription
3023xx RedirectRedirect to /admin/login?error=invalid_username_password. Invalid credentials.
3023xx RedirectRedirect to /admin/login?error=auth_err. Authorization failed for an unknown reason.
3023xx RedirectRedirect to /admin/portal. User logged in successfully.

Example

<form action="/admin/api/login" method="POST">
    <input type="text" name="username" />
    <input type="password" name="password" />
</form>

Logout User /admin/api/logout Private DELETE

Private: This endpoint is private. You must be authenticated to use it.

Description

Log a user out.



Responses

CodeTypeDescription
3023xx RedirectRedirect to /admin/login. User logged out successfully.

Example

fetch('/admin/api/logout', {
    method: 'DELETE'
})

Article Retrieval Retrieve one or multiple articles.

Retrieve Posts /posts GET

Notes:
  • Will move to /api/articles soon!
  • Articles returned from this API may be cached...

Description

Get all articles by page & type.


Parameters

NameTypeDescription
pagenumberThe page to retrieve. Each page has a max of 12 articles.
typestringThe type of articles to retrieve.

Responses

CodeTypeDescription
4004xx Client ErrorInvalid page or type.
4044xx Client ErrorNo articles found with the specified query.
2002xx SuccessSuccessfully retrieved the articles.

Example

fetch('/posts?page=0&type=Home')

Retrieve Article /admin/api/get_article Private GET

Private: This endpoint is private. You must be authenticated to use it.

Description

This endpoint retrieves a single article by ID and type.


Parameters

NameTypeDescription
idnumberThe ID of the article to retrieve.
typestringThe type of articles to retrieve.

Responses

CodeTypeDescription
4014xx Client ErrorOnly authenticated users can use this endpoint.
4044xx Client ErrorNo articles found with the specified query.
2002xx SuccessSuccessfully retrieved the article.

Example

fetch('/admin/api/get_article?id=0&type=Home')
